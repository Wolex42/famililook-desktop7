## Executive Summary

**75 failure modes** identified across 4 levels (System, Subsystem, Component, Unit). Of these:

- **6 Critical (RPN >= 300):** Single server SPOF, 0% upload conversion, CSP blocking analytics, no cross-app auth, VITE_API_KEY exposure, Hetzner memory exhaustion
- **12 High (RPN 150-299):** Currency rate drift, WebSocket memory leaks, Caddy misconfiguration, tier gating spoofable, localStorage limits, deployment branch confusion, and others
- **Remainder:** Medium and Low risk

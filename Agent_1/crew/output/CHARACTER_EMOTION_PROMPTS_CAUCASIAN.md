## HOW TO USE

1. Generate these IN THE SAME SESSION as the Caucasian base characters for style consistency.
2. Each prompt is self-contained — paste the entire block.
3. Generate at **1024x1024** on a **white background**.
4. If style drifts, add: "Match the exact same art style as the previous characters — same warm peach skin, same clothing, same line weight. Only the expression and pose change."
5. Do all 4 emotions per character before moving to the next.

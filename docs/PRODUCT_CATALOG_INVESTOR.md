## Platform Overview

FamiliLook uses AI face analysis to determine which parent a child resembles most. The analysis output — winner parent, percentage match, 8 facial feature breakdowns — drives a family of personalised physical and digital products sold direct-to-consumer via famililook.com.

**Revenue model**: Product margin on physical keepsakes + optional subscription tiers (Plus £3.99/mo, Pro £7.99/mo) + personalised message surcharge (£1.99 per item).

**Fulfilment**: Zero inventory. All physical products printed on demand via two vendor partners:
- **Prodigi** (UK) — mugs, t-shirts, prints, canvases, cushions, puzzles, bodysuits
- **QPMarkets** (CN) — playing cards, card decks, greeting cards, postcards

## Platform Architecture

```
Customer → famililook.com (Vercel)
              ↓
         AI Analysis Engine (Hetzner VPS)
              ↓
    ┌─────────┴─────────┐
    │                   │
  Prodigi API       QPMarkets API
  (mugs, prints,    (cards, decks,
   apparel, home)    stationery)
    │                   │
    └─────────┬─────────┘
              ↓
     Customer's doorstep
```

**Zero inventory. Zero warehouse. Zero staff for fulfilment.** The entire physical product line is API-driven print-on-demand.
